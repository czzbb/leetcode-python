# 206. 反转链表
**难度:easy**
## 题目
原文链接：https://leetcode-cn.com/problems/reverse-linked-list/

反转一个单链表。

**示例:**
```
输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL
```
## 思路
* 画图。一行代码就行

## 代码
```python
class a206_Solution(object):
    def reverseList(self, head):
        """
        :type head: ListNode
        :rtype: ListNode
        """
        pre = None
        cur = head
        while cur:
            cur.next, pre, cur=pre, cur, cur.next
        return pre
```
