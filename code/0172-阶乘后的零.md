# 172. 阶乘后的零
**难度:easy**
## 题目
原文链接：https://leetcode-cn.com/problems/factorial-trailing-zeroes/

给定一个整数 n，返回 n! 结果尾数中零的数量。

**示例 1:**
```
输入: 3
输出: 0
解释: 3! = 6, 尾数中没有零。
```
**示例 2:**
```
输入: 5
输出: 1
解释: 5! = 120, 尾数中有 1 个零.
```
说明: 你算法的时间复杂度应为 `O(log n)` 。


## 思路
* 尾数中会出现 `0` ，是因为 `5 * 2 = 10`
* 因此看可以分解出多少个 `5` 和 `2`；
* 只要是偶数，就可以分解出`2`，而`5` 则需要`5`的倍数；因此`2`的个数多余`5`，所以**只要看`5`的个数**就行
* 例如 `125`， 可以分解出 `（125/5）25 + （125/25）5 + （125/125）1  = 31`个`5`，
* 因为 `25`的倍数可以分解出两个`5`，`125`的倍数可以分解出`3`个`5`

## 代码
```python
class a172_Solution:
    def trailingZeroes(self, n: int) -> int:
        ans = 0
        base = 5
        while n >= base:
            ans += n//base
            base *= 5
        return ans
```
