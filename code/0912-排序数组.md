# 912. 排序数组
**难度:Medium**
## 题目
原文链接：https://leetcode-cn.com/problems/sort-an-array/

给你一个整数数组 nums，请你将该数组升序排列。

**示例 1：**
```
输入：nums = [5,2,3,1]
输出：[1,2,3,5]
```
**示例 2：**
```
输入：nums = [5,1,1,2,0,0]
输出：[0,0,1,1,2,5]
```
